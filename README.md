# 생기부 괄호·부호 제거기 (bracket-delete)

학교생활기록부(생기부) 학생 활동 엑셀에서 **모든 학생에게 공통으로 붙는 괄호·꺾쇠 ＜ ＞ 안내문구를 제거**하고, 글 속의 **모든 따옴표를 작은따옴표( ' )로 통일**하는 무료 웹 도구입니다.

👉 **바로 사용하기: https://persian32.github.io/bracket-delete/**

## 무엇을 하나요

학생 활동 엑셀(.xlsx)을 올리면:

1. 전각 ＜ ＞ 과 반각 `< >` 로 둘러싸인 작성 가이드(안내문구)를 통째로 제거합니다.
2. 큰따옴표 · 둥근 따옴표 · 전각 따옴표 · 낫표(「」『』) · 화살괄호(〈〉《》) 등 **모든 따옴표류를 작은따옴표( ' )로 통일**합니다.
3. 정리된 결과를 엑셀(.xlsx) 파일로 내려받습니다. (순번 / 이름 / 정리본 / 제거된 부호블록수)

> 학생이 실제로 쓴 글자·띄어쓰기·문장은 **그대로 보존**합니다. 부호만 정리하고, 내용은 손대지 않습니다.

## 왜 만들었나요

생기부 작성 엑셀에는 모든 학생 칸에 똑같은 작성 안내문구(예: `＜선정 이유를 쓰시오＞`)가 붙어 있어, 이를 일일이 지우는 일이 번거롭습니다. 이 도구는 그 반복 작업을 한 번에 처리합니다.

## 개인정보 — 파일은 업로드되지 않습니다

처리는 **전부 사용자의 브라우저 안(자바스크립트)에서** 이뤄집니다. 올린 엑셀 파일은 **어떤 서버로도 전송되거나 저장되지 않으며**, 창을 닫으면 아무것도 남지 않습니다. 학생 정보를 다루는 작업에 안전합니다.

## 사용 방법

1. 위 주소에 접속합니다.
2. 학생 활동 엑셀(.xlsx)을 끌어다 놓거나 파일 선택으로 올립니다.
   - '이름' 열과 '글내용(통합)' 열이 있는 형식을 기본으로 합니다. (열 이름이 다르면 화면에서 직접 지정할 수 있습니다.)
3. 부호·따옴표가 정리된 엑셀이 자동으로 내려받아집니다.

## 다음 단계 — 문장 다듬기

부호 정리 후, 결과 화면의 **"세특 다듬기 지시문 복사"** 버튼을 누르면 다듬기 지시문이 복사됩니다. 이를 사용하는 AI(Claude · ChatGPT · Gemini 등)에 정리된 파일과 함께 붙여넣으면, 교사 관찰 문체로 다듬어진 세특 문장을 받을 수 있습니다. (부호 제거는 규칙이 명확해 이 도구가 직접 처리하고, 맞춤법·문맥 다듬기는 판단이 필요해 AI가 맡는 구조입니다.)

## 바이트 기준

생기부 바이트 계산은 나이스(NEIS) 기준으로 안내합니다 — 한글 3바이트, 영문·숫자·특수문자·공백 1바이트, 줄바꿈 2바이트. (참고: https://hjh010501.github.io/neis-counter/)
